## 一、产品概述

**一句话**：用户答题测地球Online玩家评分 → 生成赛博系统界面风分享卡片 → 差评墙吐槽 → 引流个人网站。

**核心闭环**：短链接 → 首屏 → 24题测试 → 实时成就弹出 → 结果卡片(分享) → 差评墙 → 底部引流钩子

**目标人群**：18-35岁中文互联网用户，地球Online梗的消费者和参与者

**成功标准**：上线后自然传播 ≥ 5000 次页面访问，卡片分享率 > 15%

## 二、设计风格：赛博系统界面风

| 维度 | 规格 |
|------|------|
| **底色** | #060d14 深蓝黑（非纯黑，带微弱蓝色倾向） |
| **主色** | #00f0ff 霓虹青（品牌色·UI高亮·评分C+色） |
| **强调色** | #ffaa00 琥珀（成就编号·标题装饰·金句） |
| **警告色** | #ff6644 亮红（DEBUFF·负面状态） |
| **正文色** | #ffffff 纯白 + rgba(255,255,255,0.45-0.55) 次级文字 |
| **字体** | 标题：PingFang SC / Microsoft YaHei / Arial Black；命令行：monospace/Courier New |
| **UI语言** | 系统提示框 + 终端命令行 + 加载进度条 + 权限验证包裹全页 |

### 背景纹理（4层叠加）

| 层 | 纹理 | 作用 |
|---|------|------|
| 0 | 星场（12颗分布星点，radial-gradient） | 深空氛围 |
| 1 | 扫描线（3px间距，透明度0.04） | 复古终端感 |

### 英文使用规则

- ✅ 允许：命令行装饰（`ssh`、`cat`、`/var/log/`）、域名、QR标签
- ❌ 禁止：所有用户阅读内容（标题、成就名、描述、警告、金句、按钮）均为中文

## 三、页面结构与信息层级

单页 SPA，6 个 Section 切换：

```
Section 1: 首屏 — 系统弹窗 + 身份验证
Section 2: 答题页 — 终端日志样式 + 实时成就弹出
Section 3: 结果页 — 玩家评估报告（=分享卡片）
Section 4: 差评墙 — 预置差评段子瀑布流
Section 5: 引流钩子 — 底部跳转个人网站
```

### 3.1 首屏

- NASA 地球地平线照片（顶部 140px）
- 命令行装饰：`> ssh player@earth-online --eval ▮`
- 标题：「地球Online」+「系统档案·玩家评估报告」
- CTA 按钮：`▸ 点击验证身份，查看评分`
- 底部极小字：`*本系统仅为娱乐用途，最终解释权归你自己`

### 3.2 答题页（24题）

- 顶部伪终端进度条：`正在扫描玩家存档… ██████░░░ 68%`
- 每题以系统日志样式出现，选项点击后即时跳转下一题
- 成就触发时右上角滑入 toast 通知

### 3.3 结果页 = 分享卡片（1080×1920 竖版）

**7层结构**（从上到下，无重叠）：
```
L0  背景    深空 + NASA地球地平线（顶部~140px）
L1  标题    「地球Online」+「系统档案·玩家评估报告」
L2  评分    C+ 大字 + 62分 + 标语
L3  成就    3条（溢出15条→扫码查看）
L4  DEBUFF  两条警告标签
L5  金句    「没有攻略，没有重来。但今天的存档，不算差。」
L6  QR码    扫码引导 + 域名
```

**卡片必备四要素**：
1. 评分大字（3秒抓住）
2. 成就条目（10秒看完）
3. 金句（细读共鸣）
4. 二维码 + URL（回流入口）

### 3.4 差评墙

- 预置 20 条高赞差评段子，瀑布流展示
- 底部装饰性输入框（不存储数据，微信风控安全）
- 内容来源：豆瓣/B站/小红书真实高赞差评

### 3.5 引流钩子

- 底部深色条：`想要了解评分机制 / 学习制作同款H5？👉 点击访问 supersuperchik-home.pages.dev`
- 双轨策略：微信内显示文字引导复制，微信外可点击跳转

## 六、评分公式

```
总分 = 基础分 50
     + 成就加分（常见+1~3 / 稀有+4~7 / 反向+1~2）
     - DEBUFF扣分（熬夜/负债/过度通勤/社交伪装 各-2~5）
     → 最终区间 ≈ 38~85
```

| 等级 | 分数 | 称号 | 系统留言 |
|:--:|:--:|------|------|
| S | 78+ | 隐藏BOSS级 | 系统都没想到你能玩这么好——你是不是偷偷开了控制台？ |
| A | 68-77 | 高玩潜质 | 主线支线都能肝，可惜没有氪金通道 |
| B | 58-67 | 正经玩家 | 踏踏实实打主线，偶尔摸鱼，最稳定的在线用户 |
| C | 48-57 | 及格线守门员 | 能活到现在本身就及格了——没有攻略没有重来，还在线就赢了 |
| D | <48 | BUG受害者联盟 | 系统BUG的受害者，资本机制的幸存者——但你还在这本身就是一个传奇 |

## 七、技术架构

| 类型 | 方案 |
|------|------|
| **架构** | 纯静态 H5（HTML + CSS + JS），无框架，零后端 |
| **页面** | 单页 SPA，6 个 Section 切换 |
| **状态** | JS 对象管理（answers / achievements / score） |
| **卡片生成** | html2canvas 1.4.1 → Canvas → img → 长按保存 |
| **埋点** | Supabase 免费层，匿名存储分数/成就/城市（不存个人信息） |
| **PV/UV** | Cloudflare Web Analytics（内置，零配置） |
| **部署** | Cloudflare Pages（免费 + 全球CDN + HTTPS） |
| **域名** | `<项目名>.pages.dev`（Cloudflare 默认域名） |
| **字体** | 系统字体栈（PingFang SC + Microsoft YaHei + monospace），零加载 |

## 八、微信风控规避

| ✅ 做这些 | ❌ 不做这些 |
|------|------|
| 免费测试，零门槛点开即测 | 不强制分享才能看结果 |
| 长按保存图片分享（非转发链接） | 不要手机号/微信号/头像授权 |
| 底部品牌标识 ≤10%面积 | 不用「转发给X个好友」文案 |
| 差评墙假输入框（不存数据） | 不自动跳转外部App |
| 纯静态，无用户数据收集 | 不存储用户答案到服务器 |
| Cloudflare Pages 零成本抗DDoS | 不做红包/抽奖/诱导机制 |
| Supabase 匿名埋点（仅分数/成就/城市） | 不做 JS-SDK 分享（改用长按保存替代） |
| 底部链接双轨（微信内文字引导复制，微信外可点击） | |

**灰色地带**：
- 短链接 → 用 Cloudflare Pages 自带域名，避免短链被微信拦截
- 差评墙内容 → 全部预置段子，无UGC，无审核风险
